üñ•Ô∏è Option A: Same Computer

How it works: Others access the files directly

Cons: ‚úó Only works on that computer

üåê Option B: Local Network (LAN)

How it works: Run a local web server others can access

Using Python (Easy!)

  1. Open PowerShell in the HTML folder
  2. Run this command:
    python -m http.server 8000
  3. You'll see: Serving HTTP on 0.0.0.0 port 8000
  4. Others on your network go to: http://YOUR_IP:8000

How to Find YOUR_IP:

  1. Open PowerShell
  2. Run: ipconfig
  3. Look for "IPv4 Address" (like 192.168.1.100)
  4. Share this: http://192.168.1.100:8000

Pros: ‚úì Local access, ‚úì Works from other computers on WiFi

Cons: ‚úó Only while server is running, ‚úó Not accessible from internet
